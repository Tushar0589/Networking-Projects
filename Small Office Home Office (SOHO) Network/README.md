# 🌐 SOHO Network Implementation using Cisco Packet Tracer

A beginner-friendly networking project that demonstrates the design and implementation of a **Small Office/Home Office (SOHO) Network** using **Cisco Packet Tracer**. The project simulates a real-world office environment by implementing VLANs, Inter-VLAN Routing, DHCP, and wireless connectivity.

---

## 📖 Project Overview

The objective of this project is to build a secure, organized, and scalable network for a small office. The network is divided into multiple departments using VLANs, allowing efficient communication while maintaining network segmentation and security.

This project focuses on applying fundamental networking concepts commonly used in enterprise environments.

---

## 🎯 Objectives

- Design a structured SOHO network
- Configure VLANs for different departments
- Enable communication between VLANs
- Automate IP address assignment using DHCP
- Configure wireless connectivity
- Verify end-to-end network communication
- Practice troubleshooting techniques

---

## 🏗️ Network Topology

The network consists of:

- 1 Router
- 1 Layer 2 Switch
- Wireless Access Point
- Employee PCs
- Admin PC
- CEO PC
- Wireless Laptop
- Mobile Device
